**项目定位**
这是一个名为“韭菜盘 / JCP AI”的跨平台桌面股票分析应用，实际项目目录是 `E:\source\go\jcp`。核心目标是把 A 股行情、K 线、盘口、F10、龙虎榜、热点舆情、研报等数据接入到桌面端，再通过多 Agent AI 讨论室给用户做股票分析和策略建议。项目说明可见 [README.md](E:/source/go/jcp/README.md)，桌面入口在 [main.go](E:/source/go/jcp/main.go)，后端主绑定对象在 [app.go](E:/source/go/jcp/app.go)。

**基本功能**
- 股票行情：支持自选股、实时行情、分时/日 K/周 K/月 K、五档盘口、大盘指数、市场状态识别。
- 图表分析：前端使用 Lightweight Charts 展示 K 线，并支持 MA、EMA、BOLL、MACD、RSI、KDJ 等指标配置。
- AI 讨论室：用户可直接提问，也可以 `@` 指定某个专家 Agent；系统支持多 Agent 串行讨论、流式进度、失败重试、取消会议、引用回复和消息清空。
- 策略系统：支持内置策略和自定义策略，每个策略可以配置多个专家角色、工具权限、MCP 服务和独立 AI 模型。
- AI 配置：支持 OpenAI 兼容接口、Gemini、Vertex AI、Anthropic，并支持 Responses API、代理配置、连接测试、system role 探测等。
- 记忆系统：按股票隔离长期记忆，包含关键事实、最近讨论轮次、摘要压缩和相关性检索。
- 数据增强：包含 F10 全景、估值、财务、机构、行业、公告、研报、财联社快讯、热点平台、龙虎榜、板块资金流、盘口异动等。
- 扩展能力：内置 MCP 管理器，可接入外部工具服务；另有 OpenClaw HTTP 服务用于外部调用深度股票分析。
- 桌面体验：无边框窗口、自定义窗口控制、主题切换、红涨绿跌/绿涨红跌切换、面板拖拽布局持久化、自动更新。

**使用框架与技术栈**
- 桌面框架：Wails v2，把 Go 后端和 Web 前端打包成桌面应用，配置在 [wails.json](E:/source/go/jcp/wails.json)。
- 后端语言：Go 1.24.4，模块名 `github.com/run-bigpig/jcp`，依赖见 [go.mod](E:/source/go/jcp/go.mod)。
- 前端框架：React 18 + TypeScript + Vite，依赖见 [frontend/package.json](E:/source/go/jcp/frontend/package.json)。
- UI 与样式：TailwindCSS、Lucide React 图标、自定义主题上下文。
- 图表：`lightweight-charts` 用于金融 K 线图。
- Markdown/流式渲染：`markstream-react` 用于 AI 回复内容渲染。
- AI/Agent：`google.golang.org/adk`、`google.golang.org/genai`、`github.com/sashabaranov/go-openai`，并自实现 OpenAI、Responses、Anthropic 适配层。
- MCP：`github.com/modelcontextprotocol/go-sdk`，用于外部工具扩展。
- 行情数据：主行情 provider 使用 `github.com/injoyai/tdx`，并有新浪、东方财富等接口作为行情、K 线、资金流、公告等数据来源。
- 中文处理：`github.com/go-ego/gse`，用于记忆相关的中文分词和相关性检索。
- 自动更新：`github.com/run-bigpig/go-github-selfupdate`。

**代码结构**
- [main.go](E:/source/go/jcp/main.go)：Wails 应用启动、窗口配置、前端资源嵌入、panic 日志。
- [app.go](E:/source/go/jcp/app.go)：后端服务聚合层，暴露给前端调用的 Wails API 基本都在这里。
- [internal/services](E:/source/go/jcp/internal/services)：行情、配置、会话、策略、F10、研报、龙虎榜、更新等业务服务。
- [internal/adk](E:/source/go/jcp/internal/adk)：AI 模型工厂、OpenAI/Gemini/Anthropic 适配、工具注册、MCP 管理。
- [internal/meeting](E:/source/go/jcp/internal/meeting)：多 Agent 会议编排、发言、总结、重试等逻辑。
- [internal/memory](E:/source/go/jcp/internal/memory)：股票隔离记忆、摘要、关键词相关性检索。
- [internal/openclaw](E:/source/go/jcp/internal/openclaw)：外部 HTTP 分析服务。
- [frontend/src/App.tsx](E:/source/go/jcp/frontend/src/App.tsx)：主界面布局，左自选、中图表/盘口/F10、右 AI 讨论室。
- [frontend/src/components/AgentRoom.tsx](E:/source/go/jcp/frontend/src/components/AgentRoom.tsx)：AI 讨论室核心 UI。
- [frontend/src/services](E:/source/go/jcp/frontend/src/services)：前端对 Wails 后端 API 的封装。

总体上，这是一个“Go 后端数据与 AI 编排 + React 桌面前端”的 Wails 金融分析工具，项目复杂度已经超过普通行情面板，重点在多 Agent 股票分析、A 股数据工具链、策略化专家配置和本地桌面体验。
